The schedule will be fully built out by attendees the first 30 minutes of each day. Unconferences can be nebulous so here's an idea of what to expect!

Example / planned discussions

  • Library usage discussion - Who's using what, experiences, likes / dislikes. LitElement vs VanillaJS vs SkateJS, etc.
  • Theming web components - custom properties vs css variables
  • HAXcms theme layer - Experiences building custom themes
  • Light DOM vs shadow DOM - Pros & cons, lessons learned
  • HAX capable elements - Wiring a design assets to HAX, how it works
  • making hello-world - Creating a hello-world element with open-wc tooling
  • Tooling workflows - Rollup? Webpack? Polymer build? experience share and comparisons of WCFactory vs OpenWc vs others
  • HAXeditor / HAXcms UX - discussion and roadmap prioritization
  • Sakai Elements - Where they are, where they should go next
  • Monorepos vs multiple repos - How to manage WCs at scale with teams of developers and projects
  • Bundle vs unbundle - When to do one vs the other, CDN discussions
  • Anything you bring to the table! These are just suggestions based on who's going to be there!
