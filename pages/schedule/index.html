The schedule will be fully built out by attendees the first day of each event as this is an Unconference format. Because unconferences can be nebulous if you've never been so we're going to supply a rough schedule for the days as well as examples of discussions you'll be apart of.

Example / preplanned discussions

  • Library usage discussion - Who's using what, experiences, likes/dislikes. LitElement vs PolymerElement, etc.
  • Theming web components - more specifically this involves our use of custom properties to manage a theme layer and leveraging inheritance to support fallback styles
  • HAXcms theme layer - Experiences in building custom themes for HAxcms entirely out of web components
  • Light DOM vs shadow DOM - Pros & cons, lessons learned
  • HAX capable elements - Learn how to wire design assets to HAX, how it works
  • making hello-world - Creating a hello-world element with open-wc tooling
  • Tooling workflows - Rollup? Webpack? Polymer build? experience share and comparisons of WCFactory vs OpenWc vs others
  • HAXeditor / HAXcms UX - discussion and roadmap prioritization
  • Sakai Elements - Where they are, where they should go next
  • Monorepos vs multiple repos - How to manage WCs at scale with teams of developers and projects
  • Bundle vs unbundle - When to do one vs the other, CDN discussions
